# 🏠 House Price Prediction using Machine Learning

A Machine Learning web application that predicts house prices based on property details such as **Area, Bedrooms, Bathrooms, Stories, and Parking**. The project is built using **Python**, **Scikit-learn**, and **Streamlit**, providing an easy-to-use interface for instant house price prediction.

---

## 🚀 Features

* 🏡 Predict house prices instantly
* 🤖 Machine Learning based prediction
* 📊 Linear Regression model
* 🎨 Clean and responsive Streamlit UI
* ⚡ Fast and accurate predictions
* 💾 Pre-trained model using Joblib
* 📂 Easy to understand project structure

---

## 🛠️ Tech Stack

## ⚙️ Installation

```bash

cd House-Price-Prediction

pip install -r requirements.txt

python train_model.py

streamlit run app.py
```

---

## 📈 How It Works

1. Load the housing dataset.
2. Train a Linear Regression model.
3. Save the trained model using Joblib.
4. Enter house details in the Streamlit app.
5. Get the estimated house price instantly.
